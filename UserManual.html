02 · Operator Guide

How to Enter Data

Follow these five steps every day, one machine at a time. The entire process takes under 2 minutes per asset.

1
Scan the QR Code on the Machine
Open your phone's camera app. Point it at the QR code sticker on the machine. Tap the notification that appears to open the form. The form will load in your browser automatically.
💡 You can also bookmark the link. Each machine has its own unique URL with the asset code pre-filled.
2
Verify the Asset Details
The Asset Code, Description, and Previous HMR load automatically from the system. Confirm that the correct machine name is displayed. The Previous HMR field is read-only — it comes from the server.
💡 If "No previous entry" shows in the Previous HMR field, this is the first ever entry for this machine. That is normal.
3
Enter Today's HMR Reading
Walk to the machine and read the Hour Meter (the small display showing total running hours, like a car odometer). Type that exact number into the "Today's HMR" field. The system will instantly calculate Running Hours = Today − Previous.
⚠️ Today's HMR must always be GREATER than or equal to Previous HMR. HMR only increases.
4
Enter Breakdown Hours & Fuel
If the machine had a breakdown today, enter the number of hours it was down. This unlocks the Breakdown Reason section — select the appropriate cause. Then enter fuel refilled in litres (enter 0 if no fuel was added).
💡 The calculation panel shows Running Hrs, Idle Hrs, and Utilization % live as you type.
5
Review & Submit
Check the live calculation panel at the bottom of the form. Verify Running Hours and Utilization % look correct. Tap "Transmit Daily Report". A green success screen confirms your data is saved.
💡 If you scan the same machine QR again on the same day, the form opens in Edit Mode with your previous entry loaded. You can update and re-submit.
✓ DO's
  • ◆Enter HMR from machine meter every single day
  • ◆Enter 0 for fuel if no refill happened
  • ◆Report even if machine was idle all day
  • ◆Edit the existing entry if re-scanning same day
✗ DON'Ts
  • ✕Never type an HMR lower than the previous value
  • ✕Never guess or estimate the meter reading
  • ✕Never leave the fuel field empty
  • ✕Never submit twice for the same asset and date
